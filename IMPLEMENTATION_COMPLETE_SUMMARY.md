## 💡 KEY INFORMATION

### **API Endpoints Available:**

**UBI System (`/api/ubi`):**

- POST `/register-citizen` - Register new citizen
- POST `/process-monthly-payments` - Process UBI payments
- GET `/citizen/:citizenId` - Get citizen status
- GET `/payment-history/:citizenId` - Payment history
- POST `/suspend/:citizenId` - Suspend payments
- POST `/reinstate/:citizenId` - Reinstate payments
- POST `/verify-eligibility/:citizenId` - Check eligibility
- GET `/statistics` - System statistics
- GET `/health` - Health check
- GET `/welcome` - API documentation

**Education System (`/api/education`):**

- POST `/create-program` - Create education program
- POST `/enroll-citizen` - Enroll in program
- POST `/update-progress` - Update progress
- GET `/programs` - List all programs
- GET `/citizen/:citizenId/progress` - Get progress
- POST `/issue-certification` - Issue certificate
- GET `/statistics` - System statistics
- POST `/initialize-defaults` - Setup default programs
- GET `/health` - Health check

# Angel Coders Recruitment & Management System

## Mission Statement
This document outlines the framework for recruiting, hiring, and managing Angel Coders - elite software developers who help build and maintain the OSCAR BROOME REVENUE platform.

## Base Compensation
- **Starting Base Pay: $175,000/year**
- Additional benefits including equity/stake in the organization, remote work flexibility, and professional development.

## Recruitment Workflow

### 1. Candidate Sourcing
- GitHub contributors with relevant open-source work
- Referrals from existing team members
- Tech community engagement
- Professional network connections

### 2. Technical Assessment
- Code review of candidate's previous work
- Technical interview with emphasis on:
  - JavaScript/TypeScript expertise
  - System design capabilities
  - Security and compliance knowledge
  - Financial technology experience

### 3. Onboarding Process
- Complete security clearance
- Access to codebase and development environment
- Mentorship pairing with senior developers
- 90-day evaluation period

## Angel Coder Roles

| Role | Focus Area | Experience Level |
|------|------------|------------------|
| Core Developer | Platform features | Senior |
| Security Engineer | Security & compliance | Senior |
| Integration Specialist | API integrations (Plaid, JPMorgan) | Mid-Senior |
| DevOps Engineer | Deployment & infrastructure | Senior |
| AI/ML Engineer | Artificial intelligence services | Expert |

## Benefits Package
